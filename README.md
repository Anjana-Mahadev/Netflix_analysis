# Netflix Movies & TV Shows Data Analysis
This project performs exploratory data analysis (EDA) on the Netflix dataset.
It includes data loading, cleaning, visualization, and insights generation.

# Features
- Load and clean Netflix dataset  
- Handle missing values  
- Visualize genre distribution  
- Analyze movie vs TV shows  
- Analyze content added over time
